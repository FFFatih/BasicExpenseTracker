# 📊 Basic Expense Tracker

(Designed for educational purposes)
A simple personal expense tracking application built with Flutter.  
Allows users to add daily expenses and categorize them.

## 🚀 Features
- Add expenses
- View expense list
- Categorize expenses
- Simple and clean UI
